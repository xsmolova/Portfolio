01. E-shop

TheWatch - Online Shop for Watches

HTML5
CSS3
JS
PHP
Laravel

The online shop for watches was my first website. The creation process consisted of designing a data model, wireframes and subsequent frontend and backend programming. 02. Neuroevolution

Neuroevolution for Flappy Bird game

Machine Learning
ANN
RL
GA
Unity
C#

After I took a machine learning course, I worked on the development of a neural network in C#. I recreated a popular Flappy Bird game in Unity and the behavior of each bird was controlled by a neural network. The aim was to compare the speed and efficiency of learning curve in the case of ANN, but also ANN with reinforcement learning. I further optimized these neural networks with genetic algorithm.

03. Driving Simulation

Analysis of Driver Using Eye Tracking

Simulator
Tobii eyetracker
Unity
C#

My master thesis was focused on the analysis of a driver behaviour when distracted by mobile communication using eye tracking. I created a driving simulation in Unity, that exposes driver to to dangerous road situations based on the information obtained from eye tracker. The aim was to compare driver's reaction time when distracted by mobile device and head-up display.

04. MAR Application

StellAR – Augmented Reality in E-learning

iOS
AR
Vuforia
X-Code
Unity
C#

The StellAR application was the result of my bachelor's study. The aim of the application was to educate students about the solar system. I chose to develop on the iOS platform and I used Ground Plane detection from Vuforia. StellAR is a mobile augmented reality (MAR) application which can be used both inside and outside the classroom.

05. Indie Game

Moth

Unity
C#
Blender

Indie game Moth was created as a collaboration with Ing. Kristína Macková. The aim of the game is to hunt down a cryptid - Mothman and take a photo as a proof of its existence. First we designed the Game Design Document, where we defined details about the gameplay and game mechanics. Then we started developing the horror game Moth inspired by Slender. I also tried 3D modelling in Blender and made Mothman model for the game myself.
