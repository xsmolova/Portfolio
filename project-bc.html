StellAR – Augmented Reality in E-learning

Brief
The StellAR application was the result of my bachelor's study. The aim of the application was to educate students about the solar system. I chose to develop on the iOS platform and I used Ground Plane detection from Vuforia. StellAR is a mobile augmented reality (MAR) application which can be used both inside and outside the classroom.

Process
Analyze domain -> Design proposal -> Wireframes -> 3D models -> Vuforia setup -> Level development -> X-Code -> Testing

Background
StellAR was developed for the European project NEWTON, one of the aims of which is to improve the learning process and increase learning outcomes. Testing of the application confirmed that the visual content is easier for the student to remember and students prefered to learn through the application which motivated them to perform better.
