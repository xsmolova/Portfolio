05. Indie Game

Moth

Brief
Indie game Moth was created as a collaboration with Ing. Kristína Macková. The aim of the game is to hunt down a cryptid - Mothman and take a photo as a proof of its existence. First we designed the Game Design Document, where we defined details about the gameplay and game mechanics. Then we started developing the horror game Moth inspired by Slender. I also tried 3D modelling in Blender and made Mothman model for the game myself.

Process
Game Design Document -> Wireframes -> Terrain -> Game mechanics -> Mothman 3D modelling -> Animations -> Audio -> User interface

Challenges
I enjoy rogramming games, that is why during my studies I often worked with the Unity editor and the C# language. Projects like Moth taught me to work in a team, version my code on GitHub and resolve merge conflicts.
